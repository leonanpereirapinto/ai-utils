# Studies

Notes, docs, and short explanations worth keeping around while working with AI.

## Structure

- `docs/`: distilled notes from official documentation, papers, or guides.
- `explanations/`: concepts, patterns, comparisons, and implementation notes.
- `templates/`: reusable templates for creating new notes.

Templates:

- `templates/docs-note.template.md`
- `templates/explanation-note.template.md`

Keep entries short and easy to scan. Use a template from `templates/` when creating a new note. If something grows too much, split it into its own file.
